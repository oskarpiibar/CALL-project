# CALL-project
This is a DS&AI university project, where we have fine-tuned an existing T5 language correction model to be native language specific. What this means is that the user can choose a native language on our website (hosted locally) and based on the language choise a respectful language correction model will be provided. The different models have been trained on different data and in some cases provide a different correction to the same input depening on the model.

To run the model on your local machine: 
1. Firstly you need to locat the web page folder
2. Find the file called app.py
3. Run the file, wait for the Debugger to activate
4. Then you can open the web app through the http link that appears in the terminal
5. Choose a native language and click the correct button
6. The corrected text will appear in a moment with the analyzed mistakes.

It might be necessary to set up a venv and install some necessary imports to run the file
